<클라우드 컴퓨팅>

클라우드 컴퓨팅이란 클라우드라는 단어와 컴퓨팅이라는 단어를 합친 단어입니다. 먼저 클라우드라는 단어는 한국말로 
구름이라는 뜻이고 컴퓨팅이라는 말은 컴퓨터를 이용한 기술입니다. 말 그대로 구름의 특징을 이용한 컴퓨터 기술입니다. 
구름의 특징은 형태는 존재하지만 만질 수가 없고, 언제 어디서든 볼 수 있고, 누구나 볼 수가 있는 것이라고 할 수 있습니다. 
그런 특징을 컴퓨터의 기술에 접목시켜 보면 데이터를 USB나 CD같은 실물형태에 저장하는 것이 아니고 만질 수 없는 인터넷공간에 
저장을 하고, 언제 어디서든지 핸드폰, 테블릿PC, 노트북같은 각종 IT기기를 이용해서 사용이 가능하고 누구든지 볼 수 있는 것처럼 
모두가 공유를 해서 사용할 수 있는 것입니다. 이렇게 형태가 없고 언제 어디서든 데이터를 이용 가능하기 때문에 집에서 이용하던 
데이터를 다른 곳에서 이용하려 할 때에 USB나 CD같은 휴대용 저장매체를 안 가져와서 낭패를 보는 일은 없을 것입니다. 
하지만 인터넷이 안 되는 곳에서는 이용이 어렵다는 단점도 있습니다.

<IaaS> - Infrastructure as a Service

인프라 자원(서버, 스토리지, 네트워크)을 가상화된 환경에 만들어서 사용할 수 있도록 해주는 서비스

국내 : KT의 Ucloud cs, SKT의 TCloud
국외 : 아마존의 Amazon Web Service

<PaaS> - Platform as a Service

개발자가 개발을 위한 플랫폼(운영체제, 프로그램 언어)을 구축 할 필요 없이 웹에서 서비스 받는 것

국내 : 아이온 커뮤니케이션즈의 아라돈
국외 : 구글의 AppEngine, 마이크로소프트의 Azure Platform

<SaaS> - Software as a Service

소프트웨어의 기능 중에서 사용자가 필요로 하는 서비스만 제공하는 서비스

국내 : 다우기술의 트윈캠프
국외 : 구글의 Docs & Spreadsheets, 세일즈포스닷컴의 CRM

<BaaS> - Backend as a Service

모바일 어플리케이션에 특화된 클라우드 서비스로 만약 모바일 앱을 만들 때에 서버가 필요한 경우에 
서버를 자체 구축하지 않아도 서버환경에 연결되는 모바일 앱을 만들게 도와주는 서비스

국내 : KTH의 baas.io, SK플래닛의 PlanetX, 소프트웨어라이프의 PuSiL
국외 : 구글의 Place, 마이크로소프트의 윈도애저모바일서비스

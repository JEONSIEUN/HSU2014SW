파이썬(Python)은 고급프로그래밍 언어이다. 초보자부터 숙련된 전문가까지 여러 사용자들이 사용한다. 파이썬을 사용하는 기관은 교육기관,연구기관 등이 있고 점점 증가되고 있다. 순수한 프로그래밍 언어로서의 기능과 그 외에 다른 언어로 쓰인 모듈을 연결하는 풀언어(glue language)로 이용된다. 많은 응용 프로그램에서 스크립트 언어로 채용된다. 파이썬을 사용할 때에 유용한 도움말 문서가 있고, 정리도 잘 되어있어서 좋다. 유니코드문자열을 지원하기 때문에 다양한 언어의 문자도 잘 처리된다. 또한 파이썬에서는 들여쓰기를 사용하여 블록을 구분하는 문법을 가지고 있다. 다른언어와는 다르게 선택적인 것이 아니라 제한적이다. 
또 다른 언어인 펄, 루비와 자주 비교된다 루비를 알아보자
루비(Ruby)는 동적 객체 지향 스크립트 프로그래밍 언어이다 루비가 만들어 질 때 원했던 것이 쉬운 프로그래밍, 우리의 삶을 반영하는 언어, 인간중심적인 프로그래밍 등 이었다. 그래서 루비가 가지는 특징들이 있다. .순수 객체 지향언어이고 정수,문자열 등의 모든 데이터형식, 그 외의 다른 모든 것이 객체이다. 또 가독성을 중시하고 이식성이 매우 높아 여러 가지 OS에서 동작이 가능하다. 파이썬과 비교해보자 우선 파이썬은 들여쓰기를 요구하여 정형화된 문법을 요구하고 루비는 까다롭게 서식관련 문법을 정해두지는 않았다. 전세계적으로 보면 파이썬이 조금 더 많이 사용되고 루비는 일본인이 개발하여 일본에서는 루비의 인기도 높다.
사실 루비는 인터프리터형식으로 실행되는 고기능 스크립트 언어이면서 객체지향언어이고 이러한 루비의 특징을 가지고 가독성이 좋은 스크립트언어가 파이썬이기 때문에 사실상 그다지 큰 차이는 없다고 보면 된다. 
PHP(Personal Hypertext Preprocessor) 은 오픈소스 스크립트 언어이다. 웹개발과 HTML에 포함하여 사용하기에 좋다. 대부분 유닉스,리눅스에서 사용되는 언어이면서 보편적으로 사용하는 윈도우에도 설치 가능하다. 
특징들을 보자. HTML문서 안에 직접 삽입하여 사용하기 때문에 따로 실행 파일을 만들지 않아도된다.  C,JAVA등의 언어에서 문장형식을 많이 이용하기 때문에 웹문서를 보다 쉽게 작성할 수 있다. 서버의 오픈소스의 소프트웨어도 구현할 수있다. 예로는 블로깅 도구인 워드프레스와 위키백과를 구동하는 미디어위키등이 있다. 또 다른 중요한 특징들이 있다. 데이터 베이스를 지원함으로써 HTML 로는 할 수 없는 DB 검색을 할 수 있다. 또 인터프리형 언어이기 때문에 컴파일없이 실행중간에 확인하는 것이 가능하다. 그래서 수정작업 또한 편하다.  결론적으로 PHP 는 HTML 문서에 포함되어 정적인 HTML 웹페이지를 동적으로 만들어 주는 좋은 언어라고 생각한다. 
PHP 도 Javascript와 많이 비교된다.
JavaScript 는 객체기반의 스크립트 프로그래밍 언어이다. PHP처럼 웹사이트에서 많이 사용하고 응용프로그램들에 내장되어 있는 객체에도 접근이 가능하다. 컴파일이 필요없고 다른 언어들과 문법도 비슷하다. 거의 PHP와 비슷한 언어인데 PHP는 리눅스OS에서 많이 사용된고 javascipt는 윈도우에서 많이 사용된다.
